# ftp-client

A small image based in [debian:jessie](https://hub.docker.com/_/debian/) with FTP client.

Docker hub: [dockito/ftp-client](https://hub.docker.com/r/dockito/ftp-client/)

```shell
docker pull dockito/ftp-client
```
